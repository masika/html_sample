HTMLとは

HTMLは文書の書き方のひとつです。
現在はhtml5というバージョンが広く採用されています。(拡張子は.html)
HTMLは主にブラウザで表示され、制作にはコードエディター(テキストエディター)が使われます。
HTML文書を書くことや書き方をマークアップといい、このマークアップによって構造化された文書になります。

HTMLで何ができるか

  • 画像やリンクや動画・音声などを配置することができます。
  • テキストを改行、グループ分け、強調したりすることができます。

記述例

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>サンプル</title>
</head>
<body>
    <h1>これはサンプルのページです</h1>
    <div>
        divタグで囲むとそこはグループになります。<br>
        この場合囲むというのは&lt;div&gt;と&lt;/div&gt;の間に記述するということです。<br>
        改行はbrタグです。<br>
        brタグは囲む必要がありません。<br>
        <a href='https://webliker.info/46840/' target='_blank'>参考リンク</a>
        
        <p>
            pタグで囲むと段落です。
        </p>
        
        <p>
            <img src="./images/build.jpg">
        </p>
    </div>

    <ol>
        <li>テキストを書き換えてみよう</li>
        <li>リンクを書き換えてみよう</li>
        <li>画像のソースを書き換えてみよう</li>
    </ol>
</body>
</html>
ブラウザで見るとこのようになります。

上記のファイルは./sample/sample01.htmlです。コードエディターで開いて内容を編集しブラウザで確認しましょう。

備考

ところでこのページもhtmlで書かれています
ただしこのページは皆さんがよく見るページとは違ってデザインがなされていないように感じるのではないでしょうか？
実はhtmlだけではあまり見た目を変えることはできません。
実際はマークアップの仕方によって文字のサイズやリンクの色などが変わったりしますがこれはブラウザの解釈によって最低限の修飾がされているだけです。（ブラウザデフォルトCSS）
ではデザインをするにはどうすればいいのでしょうか。
ここで登場するのがCSSです。

CSSとは

CSSはHTMLの見た目を変更するための言語です。
ブラウザによって若干の違いがありますがCSS3というバージョンが広く採用されています。(拡張子は.css)
CSSもコードエディターで制作します。

CSSで何ができるか

  • html要素のサイズ、色、位置の設定
  • マウスが重なった時など、インタラクティブな見た目の変更
  • 背景画像の設定、要素の追加
  • CSSアニメーション、トランジション
  • ウィンドウサイズに合わせたデザインの変更（レスポンシブ）

記述例

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>サンプル</title>
    <style>
        h1{
            font-size: 20px;
        }

        body{
            background-color: #ccc;
        }

        .groupe{
            background-color: rgb(255,255,255);
            padding: 30px;
            max-width:300px;
            margin-right: auto;
            margin-left: auto;
            border-radius: 10px;
        }

        img{
            width: 100%;
            height:auto;
            display: block;
        }

        .headstyle{
            color:rgb(0,0,255);
        }
    </style>

    <link rel="stylesheet" href="./css/sample.css">
</head>
<body>
    <h1>これはサンプルのページです</h1>
    <div class='groupe'>
        
        <p class='headstyle'>
            CSSはhead内に書くことができます。
        </p>
        
        <p style='color:rgb(255,0,0);font-weight: bold;'>
            CSSはstyle属性に書くことができます。
        </p>

        <p class='externalstyle'>
            CSSは外部ファイルに書くことができます。(headタグ内でcssファイル読み込み)   
        </p>
        
        <p style='border: 1px solid #ddd;padding: 5px;'>
            <img src="./images/build.jpg">
        </p>


    </div>

    <ol>
        <li>headタグ内のstyleタグの値を変えてみましょう</li>
        <li>pタグのstyle属性の値を変えてみましょう</li>
        <li>./css/sample.cssの値を変えてみましょう</li>
    </ol>
</body>
</html>
ブラウザで見るとこのようになります。

備考

htmlとCSSがあれば平面上で思いつくほとんどのデザインをすることができます。
CSSはここ数年で使える機能も増えてきたのでちょっとしたアニメなども表現できます。
ただしあくまでもレイアウト等のデザインのための言語なのでユーザーの操作に合わせた処理や計算はあまりできません。

JavaScriptとは

JavaScriptはブラウザ上で動作するプログラミング言語です。(拡張子は.js)
コードエディタで書くことができます。
